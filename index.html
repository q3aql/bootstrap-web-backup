Nginx Server with Linux (Debian 10 Buster)
Clone repository:

You can download the source code of this website from here.

Also you can use git and clone the repository with the following command:

$ git clone https://github.com/q3aql/bootstrap-web-backup.git

Software used:

  • Debian: Linux-based free operating system (OS).
  • nginx: Open-source HTTP and reverse proxy server for modern operating systems.
  • PHP-FPM: PHP FastCGI implementation with some additional features.
  • aria2: Lightweight multi-protocol & multi-source command-line download utility.
  • YAAW: Yet Another Aria2 Web Frontend in pure HTML/CSS/Javascript.
  • Bootstrap: Open source toolkit for developing with HTML, CSS, and JS.
